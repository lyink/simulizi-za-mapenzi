## ğŸ“š Import Stories with Chapters

### Admin Panel:
1. Open drawer â†’ "Admin Panel"
2. Tap "Import Stories from JSON"

### JSON Format:
```json
[
  {
    "title": "My Story - Part 1",
    "author": "Author Name",
    "category": "Romance",
    "coverImageUrl": "https://your-image-url.jpg",
    "synopsis": "Story description",
    "readingTimeMinutes": 15,
    "chapters": [
      {
        "number": 1,
        "title": "Chapter 1",
        "verses": [
          {"number": 1, "text": "First paragraph..."},
          {"number": 2, "text": "Second paragraph..."}
        ]
      },
      {
        "number": 2,
        "title": "Chapter 2 - Continuation",
        "verses": [
          {"number": 1, "text": "Story continues..."}
        ]
      }
    ]
  }
]
```

### Import:
1. Paste JSON or pick file
2. Click "Preview"
3. Verify chapters
4. Click "Import"
5. Done!
